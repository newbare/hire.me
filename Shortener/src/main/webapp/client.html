Client Shortener URL

Este Cliente foi projetado com Bootstrap e Jquery.

Esta pagina é estática e não requer um servidor mas requer que a API-Shortener esteja rodando impreterivelmente na porta 8080. Se for necessário mudar a porta edite app.js para a porta desejada. Se as suas requisições não estiverem retornando será necessário instalar e ativar um plugin no seu navegador, eu usei o Chrome Allow-Control-Allow-Origin: * normalmente so ocorre quando API e Client estão em servidores diferentes.

Exemplos de Chamadas:

Create: url= http://globo.com/esporte

retrive: alias = jR

Top 10: não precisa de parâmetros

Obs: esta é apenas uma versão bem simples para chamar a API, optei em não instalar no container para simular uma situação realista.
